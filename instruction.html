Инструкция для ВКонтакте

  • Для начала необходимо перейти в пункт "Управление сообществом".
  • Чтобы получить API токен ВК – выбираем вкладку «Настройки».
  • Во вкладке «Настройки» мы видим поля с ID приложения, «Защищенный ключ» и «Сервисный ключ доступа».
  • Именно «Сервисный ключ доступа» нам и нужен, это и есть access_token, который понадобится для выполнения каждого метода.
  • Нажимаем на глазок в конце поля, чтобы открыть и скопировать значение токена.Сервисный ключ получен, стоит его сохранить.
  • Полученный ключ доступа скопируйте и сохраните так, чтобы при использовании очередного метода было удобно прикреплять его к каждому запросу к ВК API.
  • Найдите в телеграме бота с именем «@botfarther», он поможет вам в создании и управлении вашим ботом.
  • Отправьте ему «/help», и вы увидите все возможные команды, которыми может управлять бот.
  • Чтобы создать нового бота, отправьте «/newbot».
  • Следуйте инструкциям, которые он дал, и создайте новое имя для своего бота. Если вы создаете бота только для экспериментов, то имя должно быть уникальным, вы можете использовать пространство имен вашего бота, поместив свое имя перед ним в имени пользователя. Кстати, его псевдонимом может быть что угодно.
  • Поздравляем! Вы только что создали своего бота Telegram. Вы увидите новый токен API, сгенерированный для него.
  • Созданного бота необходимо добавить в ваш канал и назначить администратором, чтобы появилась возможность автоматической публикации контента.
